#siiwoo-wordpress实时弹幕插件

##插件介绍
###siiwoo是一款实时弹幕视频插件，只需把视频源地址复制到文章中，即可生成实时弹幕视频，就是这么简单赶紧试试吧！目前支持淘宝视频源！
